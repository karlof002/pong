## Controls

| Key         | Action |
|-------------|--------|
| `W`         | Move paddle up |
| `S`         | Move paddle down |
| `1`         | Set AI difficulty to **Easy** |
| `2`         | Set AI difficulty to **Medium** |
| `3`         | Set AI difficulty to **Hard** |
| `P`         | Pause / Resume |
| `R`         | Reset rally |
| `ESC`       | Quit game |
